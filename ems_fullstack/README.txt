EMS Fullstack (Spring Boot + React) minimal project.

Backend:
- Location: ems-backend
- Build with Maven (Java 17).
- Update src/main/resources/application.properties with your MySQL credentials.
- Run: mvn spring-boot:run (from ems-backend)

Frontend:
- Location: ems-frontend
- Install deps: npm install
- Run: npm start

API base: http://localhost:8080/api/employees
